☕ Mercado Coffee Bar
Mercado Coffee Bar is a modern website built for a premium coffee shop in Karachi. It is designed to give customers a smooth and high-quality experience when they look at the menu or order coffee online.

🚀 Tools I Used
Frontend: React.js and Tailwind CSS for a clean and professional look.

Backend: Node.js and Express.js to handle the website's logic.

Database: MongoDB to store data like menu items and orders.

Other Tools: Docker for easy setup and Git for version control.

✨ Main Features
Coffee Showcase: A sliding gallery on the home page that shows the best coffee drinks.

Easy Navigation: A "Category Bar" that lets users quickly find different drinks like Frappes, Matcha, or Desserts.

Mobile Friendly: The website works perfectly on phones with a simple side menu.

Fast Search: A search bar to help users find their favorite "Iced Spanish Latte" or other items instantly.

🛠️ How to Start the Project
Copy the project: git clone [https://github.com/your-username/mercado-coffee-bar.git](https://github.com/your-username/mercado-coffee-bar.git)

Install the needed files: npm install

Run the website: npm run dev
